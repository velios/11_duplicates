# Anti-Duplicator

Script find duplicate files in folder

### How to use
```bash
#Run
$ python3 duplicates.py <folder>
#Sample output
Duplicate files in <folder> folder info:

File <filename> => <filesize> Kb duplicate <n> times in next places:
>>> <filepath1>
>>> <filepath2>

Total size of duplicate files: <size> Kb
```
